## Задание 1
Обобщенные типы. Типовые переменные, <>.
### Задача 3
Текст задачи
Сравнимое.
Создайте ссылочный тип Сравнимое, гарантирующий наличие по данной ссылке метода со
следующими характеристиками:
1.Называется “сравнить”
2.Принимает объект.
3.Тип принимаемого объекта может быть изменен без изменения самого Сравнимого.
4.Возвращает целое число.
## Алгоритм решения
Сначала создаем generic-интерфейс Comparable<T>, объявляя в нем метод compare(T other), который гарантирует возможность сравнения текущего объекта с другим объектом типа T. 
Возвращая целое число(отрицательное, ноль или положительное).
Затем реализуем этот интерфейс в классе MyNumber, указывая Comparable<MyNumber> для конкретной типизации.
Определяем приватное поле Number value для хранения данных и в методе compare(MyNumber other).
Реализуем логику сравнения через Double.compare(), преобразуя оба значения к double для универсальности.
Это позволяет сравнивать любые числовые типы (Integer, Double, Float и т.д.) через один интерфейс.

Затем использовать этот метод для трех заданных сценариев:
1) преобразовывавем список строк {"qwerty", "asdfg", "zx"} в список их длин с помощью функции, возвращающей length() каждой строки
2) преобразовываем список чисел {1, -3, 7} в список абсолютных значений через Math.abs()
3) проебразовываем список массивов и находим максимум с помощья стандартной функции

### Задача 2
Текст задачи
Фильтр.
Разработайте такой метод, который будет принимать список значений типа T и объект имеющий
единственный метод test (принимает T и возвращает boolean). Верните новый список типа T, из
которого удалены все значения не прошедшие проверку условием.
Используйте разработанный метод следующим образом:
1. Передайте в метод список со значениями: “qwerty”, “asdfg”, “zx”, и отфильтруйте все
строки имеющие менее трех символов
2. Передайте в метод список со значениями: 1,-3,7, и отфильтруйте все положительные
элементы
3. Передайте в метод список состоящий из массивов целых чисел, а получите список в
котором будут только те массивы, в которых нет ни одного положительного элемента
##Алгоритм решения
Создаем универсальный метод filter, который принимает список элементов типа T.
Делаем объект-фильтр с методом test(T value), возвращающий boolean.
Пройтись по всем элементам исходного списка, проверяя каждый с помощью метода test
и добавить в новый возвращаемый список только те элементы, для которых проверка вернула true.
Это позволит отфильтровать список по любому заданному условию, поддерживаемому реализацией интерфейса TestFilter.

### Задача 3
Текст задачи
Сокращение.
Разработайте такой метод, который будет принимать список значений типа T и способ с помощью
которого список значений можно свести к одному значению типа T, которое и возвращается из
метода.
Используйте разработанный метод следующим образом:
1. Передайте в метод список со значениями: “qwerty”, “asdfg”, “zx”, и сформируйте одну
большую строку, которая состоит из всех строк исходного списка.
2. Передайте в метод список со значениями: 1,-3,7, и верните сумму всех значений исходного
списка.
3. Имеется список, состоящий из списков целых чисел, получите общеe количество
элементов во всех списках. Подсказка: решить задачу можно в одно действие или
последовательно использовать методы из 3.3.1 и 3.3.3.
Далее необходимо изменить разработанный метод таким образом, чтобы данный метод
гарантированно не возвращал null и не выбрасывал ошибок в том случае, если исходный список
пуст.
##Алгоритм решения
Разрабатываем универсальный метод reduce, который принимает список значений типа T.
Начальное (нейтральное) значение identity того же типа и бинарный оператор reducer для последовательного объединения элементов.
Метод итерируется по списку, начиная с identity, и применяет оператор к аккумулятору и очередному элементу, возвращая итоговое значение.
1)Для пустого списка гарантированно возвращает identity, что исключает null и ошибки.
2)Для задачи со строками используется identity = "" и конкатенация, для чисел — identity = 0 и сложение.
3)Сначала преобразовать каждый вложенный список целых чисел в его размер, получив список размеров.
